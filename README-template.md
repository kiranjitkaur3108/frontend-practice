## What I Learned

While building this project, I practiced:

- Centering elements using Flexbox
- Creating responsive card layouts
- Using CSS variables
- Working with box-shadow and border-radius
- Making images responsive using width: 100%
- Using media queries for responsive design

One thing I understood better was:

```css
.card {
  width: 90%;
  max-width: 320px;
}
```

This helps the card stay responsive on smaller screens while preventing it from becoming too large on desktop screens.
